# droneport
Проект посвящён сравнению двух подходов обучения с подкреплением при управлении дроном, которому необходимо находить зарядную станцию при ограниченном заряде батареи.

## Описание

В работе сравниваются два типа агентов:

* **Рациональный агент** — использует классический Q-learning без когнитивных искажений.
* **Поведенческий агент** — использует Q-learning с функцией ценности из **Prospect Theory**, моделирующей человеческое восприятие выигрышей и потерь.

Среда моделирует навигацию дрона с учётом:

* расхода энергии
* столкновений
* зарядки батареи
* ограничения по времени эпизода

## Используемые методы

* Tabular Q-learning
* Prospect Theory Value Function
* Статистическое сравнение агентов (Student’s t-test)
* Анализ траекторий и распределений метрик

## Метрики сравнения

* Доля успешных эпизодов
* Количество шагов до достижения станции
* Финальный заряд батареи
* Количество столкновений
* Энергозатраты
